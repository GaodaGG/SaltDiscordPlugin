# 增加 SPW Discord丰富状态的插件
## 用法:
1. 在 SPW 设置 → 创意工坊 → 模组管理 → 右上角导入模组 → 选择 `Discord_Rich_Presence.zip`
2. 配置插件
3. 启用并享受它
